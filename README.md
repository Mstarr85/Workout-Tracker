# Workout-Tracker

## Description 

 * A Workout Tracker app that helps user reach their fitness goals quicker when they track their workout progress.  
 * I have been provided with all front end files and starter database entries and was responsible for the backend.

## Installation Instructions

 * Clone Repo to local machine
 
 * Navigate to folder and run "npm install" in Terminal or Command prompt
 
 * Run "node server.js" in Terminal or Command prompt
 
 * Open default browser and type "localhost:3000" in navbar 
 
                     or
                     
 * Click on Heroku Deployed Link below
 
## How It Works
 
 * When the user loads the page they're given the option to create a new workout or continue with their last workout
 
 * Select Excercise Type, either Resistance or Cardio
 
 * Fill in the information for the workout
 
 * Either choose Complete to complete workout or Add Excercise to add another excercise to workout
 
 * User will be directed to the main page where all the info will be shown from workout
 
 * User can click on the word Dashboard at the top left of the page in the navbar to see graphs of their workouts through the    week

  * Mongo 
  * Mongoose 
  * Express
  * CSS
  * HTML
  * Heroku
